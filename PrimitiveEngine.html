Primitive Engine

Primitive Engine is a project I started to improve my C++ skills. Aside from C++ I am also using SDL2 for window management and rendering as well as GLM for math, that way I can focus more on gameplay features and less on math.

This project is currently a work in progress and I have a lot more plans for it.

Ideas / Thoughts

My plans for this project is to create a game that akin to Terraria, a 2D block game with the key gameplay features being building, mining and combat.

I also plan on looking into procedural world generation.
Aside from only generation a random terrain, I want to try adding biomes, caves and natural features such as trees and ore veins.
